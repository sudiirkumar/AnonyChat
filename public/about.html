AnonyChat

Private. Fast. Empheral.

What is this?

Anonymous Chat is a real-time messaging platform that allows users to join rooms without registration, send messages anonymously, and watch them disappear after a interval. It's built for privacy and simplicity.

Key Features

  • No login or user tracking
  • Real-time messaging via WebSockets
  • Auto-destructing messages after interval
  • Join rooms using a Room ID
  • File attachment support

How it Works

Users join rooms using a simple room ID and send messages instantly over WebSockets. Messages automatically disappears after a short delay—ensuring private, ephemeral communication.

Tech Stack

  • Frontend: HTML, CSS, JavaScript
  • Backend: Node.js with Express
  • WebSocket
